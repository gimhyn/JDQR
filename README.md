# 테이블 오더 + 메뉴판 최신화


### 1. 타겟

- 요식업 종사자
- 메뉴판 가격의 최신화가 안되어 불편함을 겪는 소비자

### 2. 핵심 기능

**테이블오더(손님)**

- NFC/QR을 이용해 일행들이 동시에 주문
- 게임(후)
- 간편 결제

**테이블오더(점주)**

- 음식 메뉴 편집
- 테이블이랑 QR 연결
- 위치 등록

**메뉴판 최신화**

- 지도에 음식점 마킹 및 검색
- 음식점 상세 조회시 메뉴판

### 3. 이용 기술

**FrontEnd**

- React
- Tanstack-Query
- Zustand

**BackEnd**

- Node.js
- Spring
